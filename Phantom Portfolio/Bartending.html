Bartending Bible

Bartending Bible is a full-stack web application that was entirely built from scratch. This project showcases my skills in using HTML, CSS, EJS, Express, Node.js, and MongoDB. Bartending Bible is a social media website that allows users to upload posts on how to make their favorite cocktails detailing the recipe and ingredients needed to craft it alongside its picture. The website is easy to navigate and visually stunning, providing an enjoyable user experience.

One of the key features of Bartending Bible is the home feed that shows off the latest posts and allows users to like and comment on the post. This encourages user engagement and the sharing of new and unique cocktail recipes. The website is also fully responsive, which means it can be accessed from any device with a web browser. Users can easily navigate the site from their phone, tablet, or desktop computer.

Bartending Bible is a fun and interactive way for people to connect with other cocktail enthusiasts and expand their knowledge about the art of mixology.
